🧊知识点​

这个阶段会先讲易语言编程基础中核心的东西，快速带领大家入门，循序渐进，逐步强化所学的知识点，以实例开发的方式来贯穿这些知识点，让大家学习过程中不枯燥，本阶段基本覆盖了易语言开发中常用技术知识点，掌握这些知识点以及学习的方法，基本可以开发日常生活中的应用软件了（需要涉及到编程外的知识点需要额外学习，如 网页/APP协议等），本阶段主讲老师有丰富的工具类软件开发经验，以及近10年的授课经验，最主要的是有一颗追求完美及强烈的责任心，无论在备课、讲课还是答疑交流中，相信都可以把大家带好。

3R教室的教学内容主题为TS全栈开发，围绕此主题精心设计了一套适用于远程办公求职及海外外包接单以及自由职业的技术栈（目前国内也有越来越多的企业使用这套技术栈了）。我们的后端教学以Node.js为基础，在此之上以 Nestjs 框架为主体进行讲解，当然也包含了 Node.js 的常用生态和技能。前端教学重在讲解 React，同时也包含一些常用的周边生态，比如 Tailwindcss，Nextjs 等等。具体知识点和课程表请看下面两张图

🎓如何学习​

我们通过直播+录播+文档+源代码+永久问答的方式进行教学

为了更好的让同学们吸收知识，视频已经调整为：剪辑而非直播录制，并且新增周末直播研讨会

📏教学方式​

中途加入的学员怎么办?

新课未更完前，新加入的同学，如果基础比较好，可以边看前几期的录制，文档以及源代码，同时对比新课学习，基础比较一般或没有基础的同学直接跟着新课学习就行，并且支持老师私下一对一补课。

⚡️ 使用飞书发布每周更新的2-3节视频教程

💸 每节课都有详细的文档辅助学习

💥 每节课的源代码跟随最新生态一直更新

💅 提供课程相关技术栈的永久问答服务

🧐 周六提供整天在线的实时问答(周一到周五不定时解答问题)

🧐 周日的直播茶话会用于远程求职指导与内推，外包分配及渠道拓展，自由职业与被动收入创意分享等等

🍸时间与周期​

因为新课是剪辑视频而非直播，问答是永久的，文档和代码是实时更新的，所以学习周期可以是2个月也可以是2年后，随时想学随时学，可以按自己的知识基础，开发经验与工作时间合理安排即可，学会后也可以在工作中作为查询使用。

每周六，周日 3 节课，每节45分钟

课程学习时长约为 3 个月

🍍最终的成果​

学会使用 Nestjs 开发一个强大支持高并发，消息队列，WebSocket，动态权限等功能的后端应用

可以独立开发一个 Vite + React + TS 的后台管理系统

使用 Nextjs 开发一个 SSR 网站并托管在你自己的服务器

学会使用 Docusaurus 构建一个精美的静态博客以提升求职竞争力

学会团队协作开发（git，pr，review，merge，nosion任务等）以便更好的适应远程办公

学会使用 Gitea，Drone 等构建简单的 CI/CD 平台

学会基本的 Linux 运维技能以及 PM2，Nginx，Docker，Redis，MySQL 等服务的使用

📝课程表​

老课程的录制，文档，代码等通过飞书旧群获取

如果基础比较差的同学可以先学习我们的预备文档以及通过助教的帮助来快速掌握TS语法,CSS等入门知识 课程中的学习时长只是个大概和预估，具体需要与你自己的基础知识和时间分配相关

🚀学习工具​

目前来说使用飞书发布新课视频，构建文档以及问答的效果还是挺不错的

由于一些海外同学不方便使用飞书（因为没有+86手机号），可以直接使用我们专属的Lark群来获得视频，文档和问答服务

另外我们还有专门用于教务，内推以及发包的微信群，以及备用的公共交流群

在学习完 TS 全栈开发后，教会你如何快速使用各种远程工作的工具以便入职后直接能适用
